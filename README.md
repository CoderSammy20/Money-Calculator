# Money-Calculator

A simple and effective Command-Line Interface (CLI) tool designed to help you keep track of your finances.

## Features
- **Savings Calculator:** Quickly calculate how much you need to save to reach a specific goal.
- **Budget Checker:** Easily verify if your expenses are within your defined budget.
- **Robust:** Includes error handling to ensure the program does not crash during invalid inputs.

## Prerequisites
You need Python 3.x installed to run this program.

## Installation & Usage
1. Clone the repository:
```bash
   git clone https://github.com/CoderSammy20/MoneyCalducator-.git
```

2. Navigate into the project directory:
```bash
   cd MoneyCalducator-
```

3. Run the program:
```bash
   python main.py
```

4. Follow the on-screen prompts:
   - Choose an option from the menu (e.g. Savings Calculator or Budget Checker).
   - Enter the requested values (goal amount, monthly income, expenses, etc.).
   - The program will calculate the result and display it directly in the terminal.

## Example
```bash
$ python Money.py

Welcome to MoneyCalducator!
1. Savings Calculator
2. Budget Checker
3. Exit

Choose an option: 1
Enter your Total Money: 200
```
